# PDC Sampler Condensation Theory & Bioinformatics (ASE vs PDC)

**Version**: 1.0  
**Authors**: Gao Han (School of Biological Science and Medical Engineering, Southeast University)  
**Date**: 2025-11-04

## Overview
Two code families for our manuscript: (i) condensation theory (Monte Carlo, hygroscopic growth) for the PDC sampler; (ii) bioinformatics visualizations and statistics comparing ASE vs PDC.

## Layout
- `scripts/theory/` – Fig.2 Monte Carlo and growth curves.
- `scripts/bioinfo/` – Alpha/PCoA/shared-genus/ANCOM-BC2/Wilcoxon (Figs. 4–7).
- `data/...` – inputs only (no personal data).
- `results/...` – tables exported by scripts.
- `figures/...` – publication-ready figures.
- `install.R` – one-click dependency installer.

## Use
1. Install R (≥4.0).  
2. Run `install.R`.  
3. Each script documents its inputs/outputs via repo-relative paths.
